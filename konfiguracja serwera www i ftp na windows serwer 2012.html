Konfiguracja Serwera WWW i FTP na Windows Server 2012

Aby skonfigurować serwer WWW i FTP na Windows Server 2012, należy wykonać kilka kluczowych kroków:

1. Instalacja Usług IIS

  • Otwórz „Menedżera serwera” i dodaj rolę „Usługi informacji internetowych” (IIS).
  • Wybierz dodatkowe funkcje, takie jak ASP.NET czy FTP Server, jeśli są potrzebne.

2. Tworzenie Witryny WWW

  • W IIS Managerze dodaj nową witrynę, definiując jej nazwę, lokalizację plików oraz port (domyślnie 80).
  • Upewnij się, że witryna jest uruchomiona i dostępna.

3. Konfiguracja Serwera FTP

  • Dodaj rolę FTP poprzez Menedżera serwera.
  • Utwórz nowy punkt końcowy FTP, definiując nazwę, lokalizację oraz ustawienia autoryzacji (użytkownicy i uprawnienia).

4. Zapora Sieciowa

Upewnij się, że porty (80 dla HTTP, 21 dla FTP) są otwarte w zaporze, aby umożliwić dostęp z zewnątrz.

5. Testowanie

Sprawdź dostępność witryny WWW w przeglądarce oraz połączenie FTP przy użyciu klienta FTP.

Konfiguracja serwera WWW i FTP na Windows Server 2012 jest stosunkowo prosta, a poprawna konfiguracja zapewnia stabilne i bezpieczne usługi.
